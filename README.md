# KH2FM-Bleach-Mod-PS2-v0.07
KH2FM-Bleach-Mod-PS2-v0.07

Replaces most Texts, Images and Models of Sora and Roxas with Ichigo and Zangetsu from Bleach.
Replaces all Keyblades with Zanpakutos.

Credits:
OpenKH-Team for the modding Tools and documentation
aduniiss for parts of the Title-Screen
GeminiHero for some parts of his Roxas Complete patch this is based on

Version: 0.07

How-to:
1. download all files, including "KH2FM_Toolkit.exe", "Patchmaker.bat" and "Patchmaker - Bleach Mod v0.07.txt".
2. Download Crazycats English patch https://crazycatz00.x10host.com/kh/kh2-patches.ps2/#cm
3. place all files besides a KH2FM.iso
4. double-klick Patchmaker.bat
5. press Enter
6. copy+paste the contents of "Patchmaker - Bleach Mod v0.07.txt" inside the command prompt
7. you will get an "output.kh2patch"
8. rename "output.kh2patch" to "5. Bleach Mod.kh2patch"
9. rename all .kh2patches like this:
      1. Translation.kh2patch
      2. Extras.kh2patch
      3. GoA Warp.kh2patch
      4. HDification rev255.kh2patch
      5. Bleach Mod.kh2patch
10. select all .kh2patch files and drag and drop the files ontop of "KH2FM_Toolkit.exe".
11. Press Enter
12. Enjoy!
